
 ** == ile neyi karşılaştırırız? === ile neyi karşılaştırırız?
 Cevap : == operatörü , yapısal eşitliği kontrol eder ve iki nesnenin değerlerini karşılaştırır. === operatörü ise referans eşitliğini kontrol eder ve iki nesnenin aynı bellek adresine sahip olup olmadığını kontrol eder . Örnek olarak : 

	val a: Int = 5
	val b: Int = 5
	println(a == b) // true  // == kontrolü
	**********
	val x: Int = 5
	val y: Int = 5
	println(x === y) // true veya false  // === kontrolü
