2."Akıllı Dönüşüm" (Smart Cast) ne demektir? Farklı kod örnekleri ile açıklayın. Bu özelliğin sınırlamaları nelerdir?
    fun printStringLength(str: Any) {
        if (str is String) {
            println("String length is: ${str.length}") // Smart cast: str automatically cast to String
        }
    }
    printStringLength("hello") // Output: String length is: 5

3."Güvenli & Güvensiz" operatörler nelerdir?
    safe operator - val length: Int? = nullableString?.length
    non safe operator - val length: Int = nullableString!!.length

9.- "`val result = 1L + 3`" // "Long + Int" gibi bir işlemin sonucunda "result" değişkeninin tipi ve değeri ne olur? Neden böyle olduğunu açıklayın.
    when you do any operation with the mixed type numbers,the type of result will be the type of biggest one.
    so tle type of 'result' is L(Long)

    The type of the result is Int, because of  integer overflow
11.- "`val x = 5 / 2` `println(x == 2)`" gibi bir işlemin sonucu ve tipi nedir? Neden böyle olduğunu açıklayın.
    the result is true because "==" is comparing their exact values so,2 is  equal to integer value of  '5/2' = 2

12- "`val x = 5L / 2` `println(x == 2L)`" gibi bir işlemin sonucu ve tipi nedir? Neden böyle olduğunu açıklayın.
    here 5L is Long and 2 is an Int.so the result will be Long. //True
13.- "`val x = 5 / 2.toDouble()` `println(x == 2.5)`" gibi bir işlemin sonucu ve tipi nedir? Neden böyle olduğunu açıklayın.
    here 5 and 2 is integers and to toDouble() is for convert its type to double. so the result will be True
